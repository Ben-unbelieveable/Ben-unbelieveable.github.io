抗体-02.指标-01.亲和力指标总览

发表于 2026-06-23 | 分类于 知识
抗体–抗原亲和力相关核心指标一览：KD、kon、koff、ΔGbind、EC50/IC50 的定义、测定平台、相互关系与选用原则；为指标子系列入口。

本文属 指标子系列（见 抗体-00.文档索引），汇总抗体亲和力评估中常见读数类型。结合机制与结构分工见 抗体-03.互作-01.抗体抗原互作原理介绍；(EC_{50}) 与 (K_{\mathrm{D}}) 的测定细节与换算见续篇 抗体-02.指标-02.EC50与KD的测定与换算。

段末注释：亲和力（Affinity） 指单结合位点的 intrinsic 结合强度；亲合力（Avidity） 为多价协同结合的表观增强。

1. 指标速查表

指标 类型 越小/越大越强？ 主要测定 本文档章节
(K_{\mathrm{D}}) 热力学平衡 越小越强 SPR、BLI、ITC §2
(k_{\mathrm{on}}) 动力学 越大结合越快 SPR、BLI §3
(k_{\mathrm{off}}) 动力学 越小解离越慢 SPR、BLI §3
(\Delta G_{\mathrm{bind}}) 热力学 越负越强 ITC 或由 (K_{\mathrm{D}}) 换算 §2
(EC_{50}) 表观终点 越小越强 ELISA 等 §4 → 指标-02
(IC_{50}) 表观终点 越小抑制越强 竞争 ELISA、中和实验 §4
BSA 结构 通常越大界面越大 共晶结构计算 §5

2. (K_{\mathrm{D}}) 与 (\Delta G_{\mathrm{bind}})

[
K_{\mathrm{D}} = \frac{[\mathrm{Ab}][\mathrm{Ag}]}{[\mathrm{Ab!:!Ag}]} = \frac{k_{\mathrm{off}}}{k_{\mathrm{on}}}
]

[
\Delta G_{\mathrm{bind}} \approx RT \ln K_{\mathrm{D}}
]

  • 金标准平台：SPR、BLI（实时、label-free）。
  • 适用：lead 表征、文献对比、计算模型回归标签（宜统一平台）。
  • 注意：报告时需注明抗体格式（Fab/IgG）、抗原形式、温度、pH。

3. 动力学参数

同一 (K_{\mathrm{D}}) 可对应不同「动力学性格」——therapeutic 开发需同时关注 (k_{\mathrm{off}})（靶点停留）与 (k_{\mathrm{on}})（占据速率）。详见 抗体-03.互作-02.影响亲和力的因素 §1.2。

4. (EC_{50}) / (IC_{50})（表观指标）

  • (EC_{50})：半数有效浓度（结合或效应信号达 50% 最大值）。
  • (IC_{50})：半数抑制浓度（抑制 50% 信号或活性）。

二者均为终点实验拟合值，不等于 intrinsic (K_{\mathrm{D}})（除非满足严格 1:1 结合与无试剂干扰条件）。测定、换算与报告规范见 抗体-02.指标-02；SPR/BLI 原理与平台对比见 抗体-02.指标-03。

5. 结构派生指标

指标 含义 用途
BSA 结合界面埋藏表面积 评估界面大小；与 (
(S_c) 形状互补分数 结构质量、对接排序

不参与实验剂量–反应拟合，多用于结构分析与 in silico 排序。

6. 概念三分：Affinity / Avidity / Specificity

概念 核心问题
Affinity 单一位点多紧？
Avidity 多价协同后表观多强？
Specificity 是否只认靶标？

IgG 双 Fab 测得的 (EC_{50}) 常含 avidity；Fab + SPR 更接近 affinity。

7. 选用原则（简表）

阶段 推荐指标
抗体库初筛 (EC_{50})（ELISA，高通量）
Lead 优化 (K_{\mathrm{D}}) + (k_{\mathrm{off}})（SPR/BLI）
发表 / IND SPR (K_{\mathrm{D}}) 为主，ELISA 补充
计算训练 单一平台标签；勿混 (K_{\mathrm{D}}) 与 (EC_{50})
